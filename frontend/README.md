# A Pharmacy Management System

### 🌐 **[Live Link](https://pharmacy-frontend-lt8h.onrender.com/login)**

A full-stack Pharmacy Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) to streamline pharmacy operations, medicine management, customer ordering, and administrative control.

## Features

### Admin

* Manage medicines 
* Manage staff accounts
* Manage customer accounts
* View and manage all orders
* Monitor dashboard analytics
* Approve or reject customer and staff requests
* Manage system activities

### Staff

* Manage medicines
* Process customer orders
* Update order statuses
* Assist customers
* View assigned dashboard statistics

### Customer

* Browse available medicines
* Add medicines to cart
* Place orders
* Track order status
* Manage profile and saved addresses
* Complete online payments

---

## Authentication & Verification

The system supports secure authentication using:

* Email Address

During Sign Up, a verification code (OTP) is required for account verification.

### Verification Process

1. User enters Email Address.
2. A verification code is generated by the system.
3. The verification code is sent to the registered Email Address.
4. User enters the received code.
5. After successful verification, signup proecess is completed.
6. Admin/staff approves the user account.
   
This ensures secure account authentication and prevents unauthorized access.

## Important Usage Instructions

### First Login Requirement

To use the system properly, follow the workflow below:

1. Sign in as **Admin**.
2. Create and manage **Staff** accounts.
3. Create or approve **Customer** accounts.
4. After Admin setup is completed, Staff and Customers can access all features available to their respective roles.

Without an Admin account, Staff and Customer functionalities may not be fully accessible.

## System Workflow

### Step 1

Admin logs into the system.

### Step 2

Admin manages Staff accounts.

### Step 3

Admin or staff approves Customer accounts.

### Step 4

Medicines are managed by Admin and Staff.

### Step 5

Customers browse medicines and place orders.

### Step 6

Admin/Staff process orders and update order statuses.

### Step 7

Customers track order progress and complete payments.
